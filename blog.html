Question-1: What are the different ways to select an element in the DOM?

An element can be selected by ID(getElementById), class(getElementsByClassName), tag name(getElementsByTagName), or query selectors like querySelector and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns the HTML content of an element, innerText returns the visible text content (excluding hidden elements), and textContent returns all text content including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where an event listener is attached to a parent element instead of individual child elements. When an event occurs on a child element, it bubbles up to the parent, which then handles the event based on the target element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event triggered on a child element propagates up to its parent elements in the hierarchy, allowing event listeners attached on parent elements to be triggered as well.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement(). To add it to the DOM, use parentElement.appendChild(childElement). To remove an element, use parentElement.removeChild(childElement).
